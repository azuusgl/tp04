Seguel Azul

1.¿Cuál es la etiqueta principal utilizada para crear una tabla en HTML?

La etiqueta principal que se utiliza para crearla tabla es table

2.¿Qué etiqueta se utiliza para definir una fila en una tabla?

La etiqueta que se utiliza para definir la fila seria la etiqueta "tr"

3.¿Cuál es la etiqueta utilizada para crear una celda de encabezado en una tabla?

La etiqueta que se utiliza es la etiqueta de "th" ya que define una celda como encabezado de un grupo de celdas en una tabla

4.¿Cuál es la etiqueta utilizada para crear una celda de datos en una tabla?

La etiqueta que se utiliza para crear celdas de datos en una tabla son los atributos colspan o Rowspan es el estiramiento de una celda para cubrir varias filas o columnas

5.¿Qué atributo se utiliza para especificar cuántas columnas debe abarcar una celda?

El atributo que se utiliza seria colspan pero no iria solo, va a compañado de un signo igual junto a unas comillas dentro de ello se colocaria el numero de filas que se desee realizar

6.¿Qué atributo se utiliza para especificar cuántas filas debe abarcar una celda?

El atributo que se utiliza es Rowspan, seria practicamente lo mismo que en colspan, se colocaria el signo igual y las comillas junto al numero de filas

7.Si necesitamos crear una tabla con 10 filas ¿qué etiqueta y cuántas veces se debe utilizar para definir las filas?

La etiqueta seria tr para definir una fina, serian las 10 veces para cada fila
